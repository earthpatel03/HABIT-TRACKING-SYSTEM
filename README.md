# HABIT-TRACKING-SYSTEM
The Habit Tracking System is a Python-based application that uses an SQLite database to help users manage and monitor their daily habits. It allows users to register, log in, add habits, and track their daily progress. Each habit can be marked as completed, and the system stores this data to maintain consistency records.

A key feature of this system is streak calculation, which tracks how many consecutive days a habit has been completed. It also includes a reminder system that notifies users at a specific time to perform their habits, helping improve discipline and consistency. The dashboard displays all habits along with their streaks and reminder times.

Overall, this project demonstrates how Python and database integration can be used to build a simple yet effective productivity tool. It encourages users to develop positive habits and can be further enhanced with graphical interfaces, analytics, and advanced notification systems.
